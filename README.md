# Laravel56-tdd-sample

Laravel 5.6 app with test driven development methods

## Getting Started

```bash
git clone https://github.com/NuBOXDevCom/laravel56-tdd-sample.git
```

And

```bash
composer install
```

## Running the tests

```bash
phpunit
```
